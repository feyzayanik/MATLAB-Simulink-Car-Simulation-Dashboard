# Simulink Brake Pedal Simulation

This project aims to develop an interactive car simulation dashboard using MATLAB and Simulink. It includes vehicle system modeling, dashboard visualization, and MATLAB script integration. The project is actively under development, with new simulation modules and features being added over time.
The repository includes:

* Simulink model (`.slx`)
* MATLAB scripts (`.m`)
* Workspace variables (`.mat`)

## Project Goals

* Build a functional brake pedal simulation in Simulink.
* Integrate MATLAB code with the Simulink model.
* Organize workspace variables for reproducible simulations.
* Improve proficiency in MATLAB and Simulink through hands-on practice.

## Requirements

* MATLAB
* Simulink

This project is currently under development.

Planned improvements include:

* Model optimization
* Better parameter organization
* Additional simulation scenarios
* Improved documentation and comments
* Performance analysis and visualization

## Purpose

This repository was created as a personal learning project to strengthen MATLAB and Simulink skills.
